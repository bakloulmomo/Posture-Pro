# PosturePro                

App iOS che monitora la postura del collo in tempo reale usando i sensori degli AirPods Pro.            
   
## Come funziona                                                                                        
                  
Indossa gli AirPods Pro e avvia una sessione. L'app rileva se stai dritto o curvo e ti avvisa con audio,
vibrazione e una pianta animata che cresce se la postura è buona.
                                                                                                          
## Funzionalità 

- Monitoraggio postura in tempo reale tramite AirPods Pro
- Pianta animata che cresce con la buona postura
- Audio filtrato in base alla postura                                                                   
- Feedback aptico quando si è curvi
- Statistiche di sessione e lifetime                                                                    
- Disponibile in italiano
                                                                                                          
## Requisiti    
                                                                                                          
- iOS 16+       
- AirPods Pro
- Xcode 15+

## Installazione                                                                                        
  
1. Clona la repo                                                                                        
2. Apri `PosturePro.xcodeproj` in Xcode
3. Avvia sul tuo dispositivo
